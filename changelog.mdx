## How to Upgrade

Follow these steps to update your Sai Gautham Kishtapuram Documentation instance to the latest version.

<Steps>
  <Step title="Backup Your Site" icon="database">
    Create a full backup of your current documentation repository.

    ```bash
    git clone your-repo.git backup-$(date +%Y%m%d)
    ```
  </Step>
  <Step title="Update Dependencies" icon="package">
    Run the update command in your project root.
    <Tabs>
      <Tab title="npm" icon="npm">
        ```bash
        npm update @saigautham/docs@latest
        ```
      </Tab>
      <Tab title="yarn" icon="yarn">
        ```bash
        yarn upgrade @saigautham/docs@latest
        ```
      </Tab>
    </Tabs>
  </Step>
  <Step title="Rebuild and Deploy" icon="upload-cloud">
    Rebuild your site and deploy to your hosting provider.
    ```bash
    npm run build
    # Deploy via your CI/CD pipeline
    ```
  </Step>
</Steps>

<Expandable title="Migration Guide for v1.0.0" default-open="false">
  If upgrading from pre-v1.0.0, update all `{component attributes}` to use double quotes:

  ```jsx
  // Before (single quotes - deprecated)
  <Card title='Old Title' />

  // After (double quotes - required)
  <Card title="New Title" />
  ```

  Test your site locally with `npm run dev` before production deployment.
</Expandable>
