## Overview

Welcome to the Sai Gautham Kishtapuram Documentation platform. You use this default space to organize and manage all your project documentation efficiently. The platform features a clean interface with the brand color `#3B82F6`, making it easy to create, edit, and publish docs for your projects.

<Callout kind="info">
This documentation space serves as your central hub. Customize it to fit your team's needs, from API references to user guides.
</Callout>

The platform supports MDX for rich content, including interactive components like tabs and steps. You collaborate in real-time, version your docs, and embed them anywhere.

## Getting Familiar with the Interface

Follow these steps to navigate the interface quickly:

<Steps>
  <Step title="Access Dashboard" icon="home">
    Log in to view your documentation spaces. Select the Sai Gautham Kishtapuram space.
  </Step>
  <Step title="Create Content" icon="plus">
    Click "New Page" to start writing. Choose MDX for advanced formatting.
  </Step>
  <Step title="Preview and Publish" icon="eye">
    Preview changes live. Publish when ready to share with your team.
  </Step>
</Steps>

Explore sections like sidebar navigation and search. Use the editor's preview mode to test components.

<Tabs>
  <Tab title="Web Editor" icon="globe">
    Edit docs directly in your browser. Supports real-time collaboration.
  </Tab>
  <Tab title="Markdown Import" icon="upload">
    Import existing `.mdx` files. Maintains formatting and components.
  </Tab>
  <Tab title="Embed Docs" icon="code">
    Generate embed codes for your apps.

    <CodeGroup tabs="HTML,React">


    </CodeGroup>
  </Tab>
</Tabs>
